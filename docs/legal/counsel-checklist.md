# Legal review and counsel engagement checklist

**Purpose:** This document is **not** legal advice. Founders **MUST** retain qualified counsel in each relevant jurisdiction before public token distribution, exchange listings, or broad marketing.

## When to engage counsel

- Before **any** sale, pre-sale, or “ points → token ” announcement that implies financial return.
- Before **employer, grant, or advisor** token allocations (securities / tax).
- Before **public mainnet** deployment of transferable DIR.
- Before **geo-targeted marketing** (U.S., EU, UK, Singapore, UAE, etc.).

## Suggested discussion agenda

1. **Token classification:** Utility vs security (Howey / local analogues); framing of **engagement rewards**.
2. **Issuer control:** How much the DAO/foundation controls issuance and marketing affects risk.
3. **Money transmission / VASP:** On-ramps, custodial wallets, fiat pairs.
4. **Consumer protection:** Disclosures for **loss of funds**, **slashing**, **governance risk**.
5. **Sanctions / AML:** Blocked jurisdictions, address screening for treasury payouts (if custodial components exist).
6. **IP and content:** DMCA / EU DSA-style obligations if operating public clients or default relays.
7. **Employment / contractor:** Token grants vs payroll.
